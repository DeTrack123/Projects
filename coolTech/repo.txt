Cool Tech Credential Management System - Git Repository Information

Repository: This project contains the complete source code for the Cool Tech Credential Management System

Project Structure:
------------------
Backend/
  - Node.js/Express REST API
  - MongoDB database with Mongoose ODM
  - JWT authentication with role-based access control
  - Models: User, OrganizationalUnit, Division, Credential
  - Controllers: auth, user, ou, division, credential
  - Middleware: JWT verification, role-based authorization

Frontend/
  - React application with React Router
  - Components: Login, Register, Dashboard, Credentials, UserManagement
  - Services: Centralized API communication with axios
  - Role-based UI rendering
  - Toast notifications for user feedback

Key Features Implemented:
-------------------------
✅ User authentication (register/login) with JWT tokens
✅ Three role-based access levels (normal, management, admin)
✅ Organizational units and divisions management
✅ Credential repository with CRUD operations
✅ Division-specific credential filtering
✅ User management (assign divisions, change roles)
✅ Password hashing with bcryptjs
✅ Protected routes with JWT middleware
✅ Responsive UI with gradient styling
